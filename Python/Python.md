# Python

## Utilisation de l'aide

L'aide sur le langage Python se trouve en ligne sur le site officiel `https://docs.python.org/3/`, mais il est aussi possible de trouver la description des objets et méthode via le shell intéractif.  
Lancement du shell intéractif en ligne de commande :

    python

Lancement de l'aide :

    help()
    help(str)
    help(str.center)
