Vue.js

Un petit mémo pour utiliser Vue.js.

Les bases

<div>{{ var_texte }}</div> : Affiche une variable dans une page HTML.
<div v-bind:title="var_texte"> : v-bind:attribut permet d'affecter une variable dans un attribut d'un élément HTML.
<div v-if="var_booleen"> : v-if permet d'afficher un élément ou non.
<div v-show="var_booleen"> : v-show permet de cacher un élément ou non.
<ul><li v-for="val in tableau">{{ val }}</li></ul> : v-for permet de boucler sur les éléments d'un tableau.
<button v-on:click="maFonction">Appuyer</button> : v-on:event permet de lier une fonction à un évènement.
<input v-model="var_texte" /> : v-model permet de lier une variable à un élément de formulaire.
