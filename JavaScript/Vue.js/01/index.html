Présentation de Vue.JS

Vue.JS est un framework javascript de type « front-end » (s’exécutant côté client), permettant le rendu de pages web.

v-if et v-show

Différence entre v-if et v-show. Le premier n'affiche pas du tout l'élément dans le DOM. Le deuxième le cache.
Ex : <div v-if="variable"></div> OU <div v-show="variable"></div>

Utilisation de v-if pour afficher ou non un élément.
Utilisation de v-show pour cacher ou non un élément.

v-for

On répète un élément du DOM pour chaque valeur d'un tableau.
Ex : <li v-for="personne in personnes">{ personne }</li>
